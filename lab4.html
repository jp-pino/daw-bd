¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Ambos lenguajes comparten algunas semejanzas: son orientados a objetos, pueden correr en servidores, en navegadores web, y pueden usar frameworks y librerias[2][3]. Sin embargo, son más las diferencias que hay entre ellos. Partiendo de que Java es un lenguaje de programación orientado a objetos y que JS es un lenguaje de scripting orientado a objetos podemos derivar otras diferencias. Mientras que Java se compila y corre dentro de un JVM o navegador, JS es interpretado y solo puede funcionar dentro de un navegador. También cabe destacar que JavaScript está diseñado para dar interactividad a documentos HTML mientras que las aplicaciones Java cubren necesidades mucho mas amplias.

  • ¿Qué métodos tiene el objeto Date?

    Descripción de métodos de JavaScript para el objeto Date[1].

  • getDate()

    Regresa el día del mes (en números).

  • getDay()

    Regresa el día de la semana (en números).

  • getFullYear()

    Regresa el año en 4 dígitos.

  • getHours()

    Regresa la hora (0 - 23).

  • getMilliseconds()

    Regresa los milisegundos (0 - 999).

  • ¿Qué métodos tienen los arreglos?

    Descripción de métodos de JavaScript para los arreglos[4].

  • toString()

    Regresa el un string que contiene los elementos del arreglo separados por comas.

  • pop()

    Elimina el último elemento del arreglo.

  • push()

    Agrega un nuevo elemento al final.

  • shift()

    Elimina el primer elemento del arreglo y mueve los demás una posición hacia arriba.

  • unshift()

    Agrega un nuevo elemento al principio y recorre los demás.
