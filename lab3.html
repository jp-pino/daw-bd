Lab 3: CSS

En este lab debemos agregar estilos CSS al laboratorio anterior y contestar nuevas preguntas.

Preguntas

...y respuestas
Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Al usar este modificador se ignoran todas las modificaciones subsecuentes. No es recomendable usar !important, en vez de esto se debería usar una buena jerarquía de especificidad de estilos CSS. En casos de conflictos se puede utilizar para que modificaciones generales no afecten otras específicas.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Porque esta se repetirá en el fondo tanto horizontalmente, como verticalmente. Hay que tener cuidado que se pueda visualizar de la manera esperada.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de una propiedad de estilo entre %, px y pt?

Es completamente dependiente de la aplicación en cuestión. % se usa para definir el tamaño de un elemento en función de otro (texto en funcion del tamaño de texto del <body> o secciones en relación al tamaño del viewport).

px y pt se utilizan para definir objetos de tamaño fijo, el primero según el tamaño de un pixel en la pantalla en la que se muestre el contenido y el segundo según el tamaño de texto impreso

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque al borrar todos los espacios y saltos de linea se reduce el tamaño del archivo, y consumir menos memoria es más rápido de cargar el contenido.
